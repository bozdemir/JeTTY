# JeTTY v0.12.0

The table-stakes release: **scrollback search**, **clickable links**, and the
tab smarts every terminal should have — designed from per-feature blueprints,
then adversarially reviewed (20 raw findings → 15 confirmed → all fixed)
before shipping.

## The ones you'll feel

- 🔎 **Search your scrollback.** `Ctrl+Shift+F`, type, `Enter`/`F3` to jump
  older, `Shift+F3` newer, live `3/17` counter, all matches highlighted.
  Streaming output won't yank the viewport while you're searching.
- 🔗 **Click links.** Hold `Ctrl` (⌘ works on macOS too) — the URL under the
  pointer gets underlined; `Ctrl+click` opens it. Plain-text URLs *and* OSC 8
  hyperlinks, wrapped lines included. Zero cost while you're not holding Ctrl.
- 📂 **New tabs open where you are.** `Ctrl+Shift+T` inherits the current
  shell's directory (Linux and macOS).
- 🏷️ **Tabs title themselves.** OSC 0/2 shell titles (ssh, vim, builds…) show
  up live; your manual rename always wins. The OS window title follows as
  `{tab} — JeTTY`.
- 🔔 **Know when a background tab talks.** Output = themed dot, bell = urgent
  dot; cleared the moment you look.
- 📜 **Scrollback, your size.** `scrollback_lines` (100–100k) + a Settings →
  Window cycler; applies live to every tab.
- 🖱️ **Detached windows now behave like real terminals** — selection
  (Shift+drag over TUIs too), wheel + alt-screen scrolling, middle-click
  paste, and a real scrollbar with drag and track-jump.

Plus 15 review fixes (stale-title repaints, search-state leaks, false activity
dots on resize, linear-time URL trimming, CJK-aware search bar, and more) —
full detail in the
[changelog](https://github.com/bozdemir/JeTTY/blob/main/CHANGELOG.md).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
