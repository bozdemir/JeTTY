# JeTTY v0.3.0

A customization release: **the window chrome now has its own font.**

## Highlights

- **Separate UI (chrome) font** — choose a family *and* size for everything
  outside the terminal grid: tab titles, the bottom status bar, the right-click
  menu, the Settings panel, help overlay, dialogs, and the welcome splash. It is
  fully independent of the terminal grid font, so you can run a tight monospace
  for code and a comfortable proportional sans for the UI.
- **"UI FONT" Settings section** — `Ctrl+Shift+P` → a size −/+/reset control with
  a live, true-size **"Aa"** specimen, plus a scrollable proportional-family
  picker (with a "System Sans (default)" row).
- **Zero cost, zero surprise** — a size change resizes the chrome in place (no
  fontconfig rescan) and never reflows the terminal grid; the ~0% idle path is
  untouched. The default look is identical to 0.2.0 — the feature is opt-in.

New config keys (persisted to `~/.config/jetty/config.toml`): `ui_font_family`
(default `""` = platform sans) and `ui_font_size` (10–28pt, default 16).

## Install

### Linux — one-line installer (prebuilt, no toolchain)

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or grab a `.deb` / **AppImage** from the assets below (the script verifies the
published `SHA256SUMS.txt`).

### Build from source (Linux or macOS)

```bash
git clone https://github.com/bozdemir/JeTTY.git && cd JeTTY
cargo build --release && ./target/release/jetty
```
