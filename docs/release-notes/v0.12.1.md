# JeTTY v0.12.1

Two tofu-box fixes spotted right after v0.12.0:

- **`dir` / `ls -C` columns are clean again.** The VT layer stores a literal
  tab character in the cell at each tab stop (so copying preserves tabs);
  the renderer was routing that control character to the missing-glyph
  overdraw and drawing a box. Control-char cells now render as blanks.
- **The tab-rename caret is a plain `|`** — the old `▏` (U+258F) has no glyph
  in many UI fonts and showed as a box.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
