# JeTTY v0.4.1

A config + CLI point release, with two community fixes from @YKesX.

## Highlights

- **Configurable shell.** JeTTY launches the wrong shell when your login shell
  (`$SHELL` / passwd) is bash but you actually use zsh/fish — so oh-my-zsh,
  autosuggestions and your plugins never loaded. Set it explicitly in
  `~/.config/jetty/config.toml`:

  ```toml
  shell = "/usr/bin/zsh"
  ```

  Empty (the default) auto-detects: `$SHELL` → passwd login shell → `/bin/bash`.
  Nothing is hardcoded to one shell.
- **`jetty --show` / `jetty --hide`** — explicit summon/dismiss over the
  single-instance socket, next to `--toggle`. Bind a dedicated key in your
  compositor. *(@YKesX, #4)*
- **`JETTY_GPU=high`** — opt-in env override to select the discrete GPU for
  dGPU-primary systems. The default stays the integrated GPU. *(@YKesX, #3)*

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
