# JeTTY v0.6.0

Visual effects.

## Highlights

- **CRT effect** — an optional retro-CRT look under **Settings → Effects**:
  curvature, scanlines, shadow-mask, bloom, chromatic aberration, vignette,
  scanline tint, plus animated roll / flicker / jitter. **Off by default**,
  fully tunable.
- **Caret flash & glow** — the caret briefly flashes when it moves (on by
  default — but it only animates for ~130 ms per move, so idle stays ~0% CPU),
  and an optional glow burst lights up the cursor on each keystroke.
- **Effects settings tab** — a 5th tab in the redesigned Settings panel groups
  every CRT + caret control. It scrolls (the only tab that needs to), clipped to
  the panel viewport.
- **macOS `.app` icon** — `scripts/make-macos-app.sh` bundles the Dock icon.

The default look and JeTTY's ~0% idle CPU are unchanged — CRT is off, and the
caret flash returns to a true idle the instant it finishes.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
