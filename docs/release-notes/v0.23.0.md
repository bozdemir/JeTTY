# JeTTY v0.23.0

**Speed-coherence refactor + the p10k resize fix.** A mostly-internal release
that pays down the two biggest structural debts in the render path while fixing a
long-standing, very visible bug. Behavior is preserved exactly — verified by an
adversarial code review (0 blocking) and a live proof harness.

## The visible fix: no more prompt-scatter on resize

Resizing the terminal — font `Ctrl +/-` or dragging the window corner — used to
scatter copies of the p10k / starship prompt across the screen, worst on an empty
tab. Root cause was alacritty's reflow rewrapping the full-width prompt and, on
grow, pulling prompts a prior shrink had pushed into scrollback back into view.

Now, when you're idle at a clean prompt with nothing but that prompt to lose, the
reflow is wiped and the shell repaints exactly one clean prompt. It's **content-safe
by construction**: a tab that has ever produced command output is never cleared —
it takes the normal reflow, so real output is never lost.

## The internal work

- **One paint chokepoint.** Every producer redraw now goes through a single
  per-surface `request_paint()`, guarded by a CI grep, so the ~0%-idle invariant
  can't silently regress from a stray ungated redraw. Every load-bearing
  occlusion/visibility gate is preserved verbatim.
- **One render + input core.** The main and detached windows share a single
  `render_grid_scene` (and a small shared input core) instead of a near-duplicate
  `render_detached_window` — so a fix lands once, not twice. Main-only passes
  (caret glow, CRT / Tier-B effects, the dropdown slide, mid-scene chrome) stay
  caller-injected, so a detached window gains none of them.
- **A real proof harness.** `scripts/verify-idle.sh` + a `JETTY_FRAME_LOG` present
  counter validate, on real hardware, that no final paint is dropped and that an
  occluded/hidden window with a flooding shell stays at zero frames.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
