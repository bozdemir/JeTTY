# JeTTY v0.17.0

**Speed proof & hardening.** JeTTY's whole pitch is speed — so this release stops
*claiming* it and starts *measuring* it, honestly. Designed from a verified
blueprint, then stress-tested by two design critics whose 7 blocking findings were
all about one thing: **don't ship a flattering or mislabelled number.**

## What's new

- **`JETTY_PERF_LOG=1`** — run JeTTY with this and it logs the three metrics that
  define a terminal's speed, to stderr:
  - **input latency** (keypress → frame-ready), sampled at a quiet prompt, with
    percentiles and an honest label of exactly what it includes and excludes
    (the shell-echo round trip is in; vsync-acquire, GPU submit, and display
    scan-out are called out separately);
  - **cold start** — a genuine *exec → first-frame* time (including pre-main
    loader cost), not a flattering main()-onwards slice;
  - **idle resident memory.**
  It's **zero-cost when off** — nothing touches the hot path unless the flag is set.

- **`JETTY_BENCH_CPU_ONLY=1`** for the headless `jetty-bench` — the CPU-side
  metrics (throughput, snapshot, and a clearly-labelled `pipeline_1byte_cpu`
  compute proxy — *not* "input latency") with no GPU at all.

- A **non-blocking CI perf report** runs the CPU-only bench on every push and
  prints the numbers. (Hard regression gating comes once we've learned the CI
  runner's real variance — gating on a noisy shared runner just produces false
  failures.)

## Honesty note

`docs/perf-budget.md` now has **real numbers measured on real hardware**, with
labels that don't oversell. The old **154 MB/s** throughput figure didn't
reproduce (**~118 MB/s median**) — it's corrected with a note, and the ≥150 MB/s
target is openly marked *unmet, under investigation*. Anything that needs the
live windowed app is marked "read live" with the command — never invented. (This
is the same principle behind v0.15 removing a fabricated perf-HUD string.)

Full detail in the
[changelog](https://github.com/bozdemir/JeTTY/blob/main/CHANGELOG.md).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
