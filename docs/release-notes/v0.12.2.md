# JeTTY v0.12.2

Follow-up to v0.12.1's tofu fixes: the whole class is now closed.

- **Chrome/UI text falls back across fonts.** Tab titles (shell-controlled via
  OSC since v0.12.0), search queries, and rename buffers can contain emoji,
  CJK, or any symbol your UI font lacks — they now render as real glyphs
  instead of tofu boxes. Zero idle cost.
- **Wide titles can't overflow tabs** — truncation counts display width
  (CJK/emoji = 2 cells) in the main tab bar, the rename editor, and detached
  window title bars.

(v0.12.1, folded in here for anyone skipping: `dir`/`ls -C` tab-stop cells and
the tab-rename caret no longer render as boxes.)

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
