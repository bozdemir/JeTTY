# JeTTY v0.13.0

The **text-attributes** release. JeTTY used to silently drop bold, italic, and
underlines — now it renders them all, correctly and without touching the ~0%
idle budget. Designed from a verified blueprint, stress-tested by two design
critics, then adversarially reviewed before shipping.

## What you'll see

- **Bold and italic** — real font faces (bold-italic too). And the thing that
  makes it safe: glyph advances snap to the cell grid, so **bold can never push
  a column out of alignment**, whatever font you use.
- **Every underline style** — single, double, dotted, dashed, and **undercurl**,
  with **per-run colors** (SGR 58). Your nvim / LSP / spellcheck squiggles now
  show up in red (or whatever color they ask for).
- **Strikethrough.**
- **Links finally underline.** v0.12 made URLs Ctrl-clickable but never drew the
  underline — now it's there, for both plain URLs and OSC 8 hyperlinks.
- **Cursor shape** follows your shell (`DECSCUSR`) — block in the shell, beam in
  vim insert mode, underline where apps ask for it, and a hollow block when the
  window loses focus.

Blink (SGR 5/6) stays intentionally unsupported (the VT engine drops the bit and
a blink timer fights the idle-CPU design — same call as ligatures). Bold/italic
on CJK/emoji fallback glyphs render regular for now.

Full detail in the
[changelog](https://github.com/bozdemir/JeTTY/blob/main/CHANGELOG.md).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
