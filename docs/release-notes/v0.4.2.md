# JeTTY v0.4.2

A small discoverability release for the **Shift+drag** text selection added in
0.4.0.

## Highlights

- **Contextual hint.** Inside an app that grabs the mouse (Claude Code, vim,
  htop, tmux), a plain drag goes to the app — so you couldn't select text without
  knowing to hold **Shift**. Now, the moment you drag without Shift in such an
  app, JeTTY shows a brief centered toast: *"Hold Shift while dragging to select
  text."* Throttled and self-dismissing — no idle-CPU cost.
- **Documented everywhere.** Shift+drag is now in the in-app Help overlay (the
  "?" button) and the README keybindings table, so it's discoverable without
  trial and error.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
