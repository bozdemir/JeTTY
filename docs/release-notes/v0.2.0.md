# JeTTY v0.2.0

A polish + correctness release. The UI got an elegant pass, the perf HUD moved to
a proper status bar, and a deep multi-agent audit (89 agents) fixed a wave of
issues — including several that make JeTTY a **correct** terminal for TUIs
(vim / htop / fzf), not just a fast one.

## Highlights

- **Redesigned tab bar** — frameless and modern (Safari / Zed / Arc style): the
  active tab is a soft, theme-derived pill (no per-tab borders or markers),
  inactive tabs are dim text only. Tab titles now render in the platform's
  **proportional sans-serif** for an elegant, non-"code" look.
- **Bottom status bar** — the live perf HUD (ms · fps · CPU% · VT MB/s) now lives
  in a slim status bar at the window bottom, off the tab row.
- **Real TUI keyboard support** — Home/End/Delete/Insert/F1–F12, `Ctrl/Shift/Alt`
  + Arrow (word-jump, selection), and Shift+Tab now send the correct xterm
  sequences. vim, htop, less, fzf and readline line-editing work properly.
- **macOS fixes** — window transparency renders correctly on Metal, and Option-key
  composed glyphs (©, ü, …) reach the shell.
- **~0% idle CPU restored** — a debounced resize no longer spins the event loop.
- **No more zombie processes** — closed shells are reaped on tab close / exit.
- **HiDPI/Retina** — chrome width math uses the measured font advance, so menus,
  the HUD and the settings panel no longer overflow on scaled displays.

Plus: dropdown re-summons on the last-used monitor, lazy GPU offscreen allocation,
hardened single-instance IPC, phosphor shader fixes, and a full docs pass
(`CONTRIBUTING.md`, `CHANGELOG.md`, issue/PR templates).

## Install

### Linux — one-line installer (prebuilt, no toolchain)

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or grab a `.deb` / **AppImage** from the assets below (the script verifies the
published `SHA256SUMS.txt`).

### Build from source (Linux or macOS)

```bash
git clone https://github.com/bozdemir/JeTTY.git && cd JeTTY
cargo build --release && ./target/release/jetty
```
