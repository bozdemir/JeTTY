# JeTTY v0.10.0

A correctness + hardening release. A whole-codebase multi-agent audit
(22 finders → 149 adversarial verifiers) confirmed 79 issues; **68 were fixed**
and the full diff was re-reviewed for regressions before shipping. Nothing you
rely on changes — things that used to crash, leak, or misrender no longer do.

## Highlights

- **Paste is safe again.** A crafted clipboard payload can no longer smuggle a
  bracketed-paste end marker to break out of the paste guard and run commands.
- **Output floods can't freeze the UI.** `yes`, `cat huge.log`, or a runaway
  build feed the terminal in bounded slices — input and redraw stay responsive
  and Ctrl+C still reaches the shell.
- **Your settings can't be silently wiped.** A malformed `config.toml` is kept
  as `config.toml.bad` and defaults load in memory, instead of the file being
  overwritten with defaults.
- **Detaching a tab can't crash the app.** If the OS window or GPU context
  fails, the tab stays put and the detach is cancelled — no more killing every
  shell.
- **Text stops silently blanking out.** The glyph atlas is trimmed again (it
  used to grow until full, then stop rendering); wide CJK keeps columns aligned.
- **Sharper on HiDPI.** Settings panel, context menu, and help overlay scale
  with the UI font on 2×/fractional displays instead of overlapping.
- **Shells are reliably reaped** on tab close, even when a background job holds
  the PTY open (wait-based reaper with a SIGHUP → guarded-SIGKILL escalation).

Plus: OSC 10/11/12/4 color queries follow the live theme, SGR 8 concealed text
is hidden, a per-user `0700` IPC socket, hardened GPU surface/texture sizing,
non-QWERTY / Cmd / Shift+Insert / modified-PageUp keyboard fixes, detached-window
mouse-wheel + scale-factor handling, multi-monitor tear-out, and shell-script
quoting/timeout hardening.

The tree ships clean: `cargo build --release`, **297 tests pass**, and
`cargo clippy --workspace --all-targets` reports **0 warnings**.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
