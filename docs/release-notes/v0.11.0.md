# JeTTY v0.11.0

A deep correctness + hardening release. A 124-agent audit surfaced **40 bugs**,
each independently double/triple-verified; all were fixed and then
regression-reviewed before shipping.

## The ones you'll feel

- **No more frozen tabs.** A synchronized-update (DEC 2026) sequence — e.g. from
  `kill -9`-ing nvim/zellij mid-redraw — could hang a tab indefinitely. Fixed.
- **A bad `shell` in your config can't brick startup** anymore — it falls back
  and tells you, instead of exiting before a window shows.
- **Mouse wheel scrolls `less`/`man`/`git log`** (alt-screen), and **mouse
  drag works in tmux/nvim** (motion reports).
- **macOS Cmd shortcuts work** (Cmd+C/V/A/W/…), and **font/opacity hotkeys work
  on Turkish-Q / QWERTZ** and other layouts.
- **Hidden/minimized windows go back to ~0% CPU** — no invisible-frame render
  burn from effects or PTY output.
- **Detached windows** got live font propagation, text selection, click
  forwarding, type-to-bottom, and their advertised shortcuts.
- HiDPI drag-and-drop + Effects-scroll fixes, bounded PTY write queue (no OOM
  from query floods), symlink-safe config saves, spec-correct autostart escaping,
  and ~a dozen more edge cases.

Full detail in the [changelog](https://github.com/bozdemir/JeTTY/blob/main/CHANGELOG.md).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
