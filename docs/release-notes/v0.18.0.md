# JeTTY v0.18.0

**The command palette.** One chord, and everything JeTTY can do is a keystroke
away. Designed from a verified blueprint, stress-tested by two design critics,
adversarially reviewed, and screenshot-verified.

## What's new

Press **`Ctrl+Shift+P`** (⌘⇧P on macOS) and a themed, fuzzy launcher drops in:

- **type to filter** ~50 actions — `new tab`, `theme dracula`, `toggle crt`,
  `jump to prompt`, `open settings`, `increase font`, `detach tab`, `hide`,
  `quit`…;
- **`Up`/`Down`** to move, **`Enter`** to run, **`Esc`** to close;
- matched characters are highlighted, the list scrolls, and every one of the
  22+ themes and your open tabs are right there to jump to.

Every entry runs the *same* code the menus and shortcuts already use — nothing
duplicated — and the palette only exists while it's open, so idle stays at zero.

**Note:** `Ctrl+Shift+P` used to open Settings. Settings is still a keystroke
away — **`Ctrl+Shift+O`** or **`Ctrl+,`** (⌘, on macOS) — and there's an
**"Open Settings…"** entry in the palette.

Full detail in the
[changelog](https://github.com/bozdemir/JeTTY/blob/main/CHANGELOG.md).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
