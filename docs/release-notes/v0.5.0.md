# JeTTY v0.5.0

A Settings redesign.

## Highlights

- **Tabbed Settings.** The Settings dialog grew long (14 sections). It's now
  **4 tabs** — **Look** / **Fonts** / **Window** / **Shell** — so you see only
  what you came for, and the panel is half the height it was.
- **Shell picker.** Under the **Shell** tab, a `‹ … ›` selector lists the shells
  installed on your system (from `/etc/shells`) and lets you pick one — no config
  file editing. "System default" auto-detects (`$SHELL` → passwd → `/bin/bash`).
  Still settable via the `shell` config key too.
- **Copy clears the selection.** After Ctrl+Shift+C or right-click → Copy, the
  selection highlight no longer lingers (copy-on-select still keeps it).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
